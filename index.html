Entry for Week 2

What I’ve done this week

  • Studied the App Inventor development workflow
  • Forked and cloned the App Inventor sources, and run my own server locally(On Ubuntu 13.10)
  • Completed assignment 1: the App Inventor Stock Quotes Tutorial and Food Explorer Tutorial using the local version of App Inventor
  • Setted up Eclipse for App Inventor development
  • Completed assignment 2: modify the Camera components to allow user to toggle the camera light

Problems I’ve encountered

This week things were going just fine except a little trouble I’ve had when I was trying to build the App Inventor on my laptop. I’m using a 64bit Ubuntu 13.10 and JDK. Thus I had to install ia32-libs for AAPT to work. However, ia32-libs is no more available in 13.10. At last I found a solution on Stack Overflow(http://stackoverflow.com/questions/19523502/androids-aapt-not-running-on-64-bit-ubuntu-13-10-no-ia32-libs-how-can-i-fix). Instead of adding 32bit compatibility, I just installed libc6-i386, lib32stdc++6, lib32gcc1, lib32ncurses5 and lib32z1. Then I built the source again and 2 minutes later the message BUILD SUCCESSFUL showed up.

Plan for the next week

  • Build a weather forecast application to demo at the hackathon. The app may use Weather Underground API or WeatherBug API.
  • Continue to study the weeks 3-5 courses.
  • Make a choice between Components and Designer.
